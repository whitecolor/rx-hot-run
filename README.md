> Cycle.run for [RxJS v4 Hot](https://github.com/whitecolor/rx-hot)

## Installing


`npm install rx rx-hot rx-hot-run -S`

Note: `rx` & `rx-hot` are **required** dependencies. Without them, nothing will work.
